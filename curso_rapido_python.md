# Curso rápido de Python para profissionais do Direito (com apoio de IA)

## José Eduardo S. Pimentel

---

## 1. Introdução
- O profissional do Direito deveria saber programar?
- Por que o Python?
- O que dá para fazer com Python?
- Como o Python funciona?
- Método: programando como um "Lego".
- Como a IA vai ajudar no meu aprendizado?

---

## 2. Instalação e configurações iniciais
**Escopo**: Preparação do ambiente de trabalho, configurando o interpretador e ferramentas básicas para execução do código.

- Download do Python (versão 3.x).
- Configuração de PATH.
- IDEs e Editores (IDLE, Colab e VS Code).
- Verificação via terminal e "Hello, Word!"
- Características da linguagem.

---

## 3. Lógica de Programação
**Escopo**: Fundamentos da escrita de algoritmos e como o computador processa instruções usando a sintaxe Python.

- Tipos de dados (int, float, string, bool).
- Estruturas de dados: listas, dicionários, tuplas e conjuntos (Sets).
- Variáveis e operadores.
- Estruturas de controle (if, elif, else).
- Laços de repetição (for, while).
- Tratamento de erros (try, except)
- Funções e organização do código.

---

## 4. Baterias Inclusas (Standard Library)
**Escopo**: Exploração das ferramentas nativas do Python que permitem realizar tarefas complexas sem instalações externas.

- Manipulação de arquivos.
- Módulos `os` e `sys`.
- Módulos `datetime` e `re`.
- Módulos `math`, `random` e `hashlib`.
- JSON e estruturação de dados.

---

## 5. Criação de Ambientes Virtuais
**Escopo**: Técnicas de isolamento de dependências para manter projetos organizados e evitar conflitos de versões.

- Conceito de isolamento.
- Módulo `venv`: criação e ativação de ambientes.
- Gerenciamento com `requirements.txt`.

---

## 6. Bibliotecas Terceiras
**Escopo**: Expansão das capacidades da linguagem através do ecossistema de pacotes da comunidade.

- O gerenciador `pip` e PyPI.
- pypdf: manipulação de arquivos PDF.
- python-docx: manipulação de documentos Word.
- LiteLLM: interface unificada para modelos de linguagem (LLMs).
- Requests: consumo de APIs.
- Beautiful Soup: extração de dados de arquivos HTML e XML.
